Инструкция

Здравствуйте дорогие пользователи!!!

1. В окнe ДАТА вписать дату или выбрать сегодняшнее число.

2. В окно ПОЛУЧЕНО внесите колличество полученных денежных средств.

3. В окно ПОТРАЧЕНО внесите потраченныю сумму. (Если нечего — то пишите 0)

4. В окно НА ЧТО ПОТРАЧЕНО записывается цель трат. (Данная строка является не обязательной к заполнению)

5. В окно ОСТАЛОСЬ нажать на кнопку (========).

6. Для создания новой строки таблицы нажать кнопку ДОБАВИТЬ. (Данная кнопка находится под таблицей)
